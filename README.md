🛠️ 개발 요구 사항

 - 모바일, 패드, 웹을 지원하는 반응형으로 제작한다. 
 - 메뉴 클릭시 해당 목차로 이동한다.
-  작품 번호를 클릭했을시 해당 작품의 해설로 이동한다.
- 길찾기 바로가기 링크를 제공한다.
- 전시장 위치를 지도로 표기한다.
